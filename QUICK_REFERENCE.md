# Quick Start Guide for Teachers

## 🔐 First Time Setup

1. Open Gear Kiosk in your browser
2. Click **"Create New Classroom"**
3. Choose a username (e.g., your name or class code)
4. Create a 4-digit PIN (this is your master PIN - remember it!)
5. Click **"Create Account"**

## 📝 Daily Workflow

### Login
- Username: _______________
- PIN: ______ (your 4-digit PIN)

### Add New Equipment
1. Click ⚙️ (settings icon)
2. Enter your PIN
3. Fill in: ID + Name
4. Click **+** button

### Add One-Time Equipment (Borrowed/Temporary)
1. Click ⚙️ (settings icon)
2. Enter your PIN
3. Fill in: ID + Name
4. ✅ Check "One-time item"
5. Click **+** button
**→ Auto-deletes when returned!**

### Add Students
1. Click ⚙️ (settings icon)
2. Enter your PIN
3. Fill in: Name + 4-digit PIN
4. Click **+** button

### Add One-Time Students (Guests/Temps)
1. Click ⚙️ (settings icon)
2. Enter your PIN
3. Fill in: Name + 4-digit PIN
4. ✅ Check "One-time student"
5. Click **+** button

## 👥 For Students

### Checkout
1. Tap item you want
2. Enter your 4-digit PIN
3. Done! Item shows as OUT

### Return
1. Tap item you're returning
2. Enter your 4-digit PIN
3. Done! Item shows as CHECK-IN

## ✓ Teacher Verification

When items are returned, they need verification:
1. Click **"Verify All Pending"** button (top right)
   OR click individual items marked CHECK-IN
2. Enter your teacher PIN
3. Items return to READY status

**Note:** One-time items skip this step and auto-delete!

## 💡 Quick Tips

- **One-time items** = Use for borrowed/temporary gear
- **One-time students** = Use for guests or temporary access
- **Your PIN** = Used for both admin access AND verifying returns
- **Student PINs** = Can be simple (1234, 5678, etc.) since they're per-classroom
- **Logout** = Click 🚪 icon when done

## ⚠️ Remember

✅ Your PIN controls everything - keep it safe!
✅ One-time items disappear after return
✅ Regular items need your verification after return
✅ Each classroom is completely separate

## 🆘 Need Help?

**Forgot PIN?** → Contact admin
**Item stuck?** → Check status in admin panel
**Wrong checkout?** → Have student return item, then you verify
